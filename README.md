## Search

* load csv files with get_portal_datasets.py script
* grep $PATTERN \*

For example, the following queries return the rows associated with the 2nd district of Vienna:
			 grep ";1020;" *
			 grep ";90200;" *
			 grep -i ";Leopoldstadt;" *
